# login_Page
This Is the Login Page For Our Idea EventHub.That will Take the Email of the User and send OPT in the Email For Verification
A simple and responsive **Email OTP Verification system** built using **HTML, CSS, JavaScript**, and **EmailJS**.  
This project allows users to enter their email, receive an OTP, and verify it..

## 🚀 Features

- 📩 Send OTP to user email
- 🔐 OTP verification system
- 🎨 Clean and responsive UI
- ⚡ Fast client-side validation

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- EmailJS
